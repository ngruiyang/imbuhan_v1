Awalan (prefixes)

PrefixFunctionExamples
meN-Active transitive/intransitive verbtulis → menulis (to write)
baca → membaca (to read)
ber-State, possession, intransitive verbjalan → berjalan (to walk)
kerja → bekerja (to work)
ter-Accidental, passive, superlativejatuh → terjatuh (accidentally fell)
baik → terbaik (best)
di-Passive voice (by third person)beli → dibeli (was bought)
makan → dimakan (was eaten)
pe-Agent noun (doer of action)tulis → penulis (writer)
nyanyi → penyanyi (singer)
per-Causative or comparativebesar → perbesar (to enlarge)
ke-Ordinal numbers; some nounstiga → ketiga (third)

Akhiran (suffixes)

SuffixFunctionExamples
-anNoun from verb/adjectivemakan → makanan (food)
ajar → ajaran (teaching)
-kanCausative or benefactivebuka → bukakan (open for someone)
bersih → bersihkan (to clean)
-iRepeated or directed actiontanda → tandai (to mark repeatedly)
duduk → duduki (to occupy)

Apitan (circumfixes)

CircumfixFunctionExamples
ke-…-anAbstract noun, state, accidental passivemaju → kemajuan (progress)
hujan → kehujanan (caught in rain)
pe-…-anPlace, process, or abstract nounbelajar → pembelajaran (learning)
latih → pelatihan (training)
per-…-anAbstract noun, place of actiondagang → perdagangan (trade)
main → permainan (game)
me-…-kanActive causative verbbersih → membersihkan (to clean)

meN- spelling rules

Root starts withForm of meN-Example
b, p (→ silent)mem-baca → membaca; pukul → memukul
c, d, j, t (→ silent)men-cuci → mencuci; tulis → menulis
g, k (→ silent), hmeng-goreng → menggoreng; kira → mengira
s (→ silent)meny-sapu → menyapu; simpan → menyimpan
l, m, n, r, w, y, vowelme-layan → melayan; ambil → mengambil
